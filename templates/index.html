1. Mathematical Discussion

LU Decomposition is a matrix factorization method where:

A = LU

When partial pivoting is used, the equation becomes:

PA = LU
  • P = Pivot Matrix
  • L = Lower Triangular Matrix
  • U = Upper Triangular Matrix

The Doolittle Method assumes that all diagonal entries of matrix L are equal to 1.

This method is widely used in Numerical Methods for:

  • Solving systems of linear equations
  • Matrix inversion
  • Engineering computations
  • Scientific computing

2. Worked Example 1

A =

| 2  -1  -2 |
| -4  6   3 |
| -4 -2   8 |

Using Doolittle LU Decomposition: 

L =

| 1   0   0 |
| -2  1   0 |
| -2 -1   1 |

U =

| 2 -1 -2 |
| 0  4 -1 |
| 0  0  3 |
