Doolittle Method

1. Mathematical Discussion

LU Decomposition is a matrix factorization technique where a square matrix A is decomposed into:

A = LU

Where:

  • L = Lower Triangular Matrix
  • U = Upper Triangular Matrix

In the Doolittle Method, the diagonal elements of matrix L are always equal to 1.

L = \[ \begin{bmatrix} 1 & 0 & 0 \\ l_{21} & 1 & 0 \\ l_{31} & l_{32} & 1 \end{bmatrix} \]

This method is commonly used in solving systems of linear equations, inverse matrices, and numerical analysis problems.

2. Worked Examples

Example 1

Given:

A = [ 2 -1 -2 ]
[ -4 6 3 ]
[ -4 -2 8 ]

Using Doolittle LU Decomposition:

L = [ 1 0 0 ]
[ -2 1 0 ]
[ -2 -1 1 ]
U = [ 2 -1 -2 ]
[ 0 4 -1 ]
[ 0 0 3 ]

Example 2

Given:

A = [ 1 2 3 ]
[ 2 5 8 ]
[ 3 8 14 ]

After decomposition:

L = [ 1 0 0 ]
[ 2 1 0 ]
[ 3 2 1 ]
U = [ 1 2 3 ]
[ 0 1 2 ]
[ 0 0 1 ]
