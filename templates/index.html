Mathematical Discussion

LU decomposition factors a square matrix A into:

A = LU

Where:

  • L = Lower Triangular Matrix
  • U = Upper Triangular Matrix

In Doolittle Method, the diagonal entries of L are always 1.

Worked Example 1

A = 
2  -1  -2
-4  6   3
-4 -2   8

Result: 

L =
1   0   0
-2  1   0
-2 -1   1

U =
2  -1  -2
0   4  -1
0   0   3
